# Soluciones docentes  
## Proyecto: Detective de Funciones

Este apartado contiene **orientaciones pedagógicas y respuestas esperadas** para cada uno de los casos del proyecto Detective de Funciones.

Las soluciones no buscan entregar resultados cerrados, sino **apoyar la mediación docente**, destacando los focos conceptuales, posibles errores frecuentes y criterios de logro.

Este material está destinado exclusivamente al uso docente.
